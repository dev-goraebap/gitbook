# 🍰 nest.goraebap

nestjs를 사용하면서 자주 사용되는 모듈과 기능들을 npm package 로 관리하는 목적으로 만들었습니다.

생각하고 있는 주 기능들은 Oauth2 인증 및 jwt 인증에 대한 재사용성 모듈을 제공하고, 그 외에도 도메인간의 이벤트 통신 모듈, 이메일 지원, 파일 업로드, 로깅 등의 모듈등을 제공할 예정입니다.🍩

### 설치

```bash
npm i nest.goraebap
```
